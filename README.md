
# Gulp-Build-Scripts

这是一个使用 Gulp 整合的，便于快速使用的自动工作流脚本组。用于在编码过程中对代码进行质量控制、监控文件变更并处理、处理预编译语言等。

## 安装 Install

使用以下命令下载并安装依赖项：

```shell
  git clone https://github.com/seanhuai/gulp-build-scripts.git
  cd gulp-build-scripts 
  npm install
```
